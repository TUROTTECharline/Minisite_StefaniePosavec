Pour le développement web

Pour développer mon site web, je me suis servie de deux sources majeures.

  • Cours sur “OpenClassroums” : J’avais une très faible connaissance du développement web HTML/CSS. Pour combler ce manque et mener à bien ce projet, j’ai suivi le cours “Créez votre site web avec HTML5 et CSS3” afin de renforcer ces diverses compétences.
  • Intelligence artificielle "Gemini" : J’ai utilisé cette IA comme assistant. En effet, cette dernière m’a été utile pour le “JavaScript” et les éléments interactifs de mon site, les “Media Queries” pour le design responsive, mais également pour résoudre certains problèmes de codage que j’ai pu rencontrer durant le développement de mon site.
